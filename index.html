HTML Head Checklist

Hướng dẫn cấu hình phần head chuẩn cho trang web

1. Meta Charset

Khai báo bảng mã UTF-8 để hiển thị tiếng Việt đúng cách.

<meta charset="UTF-8">

2. Meta Viewport

Giúp trang hiển thị phù hợp trên thiết bị di động.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

3. Title

Tiêu đề hiển thị trên tab trình duyệt.

<title>Tên trang</title>

4. Liên kết CSS

Liên kết file CSS ngoài để tạo kiểu cho trang.

<link rel="stylesheet" href="css/style.css">

5. Liên kết JavaScript

Đặt script cuối body để trang tải nhanh hơn.

<script src="js/script.js"></script>

6. Bình luận

Dùng bình luận để chia khu vực mã rõ ràng.

<!-- Ghi chú tại đây -->

7. Thụt dòng

Thụt dòng giúp nhìn rõ quan hệ cha-con giữa các thẻ.
